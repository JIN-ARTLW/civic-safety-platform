시민 참여형 스마트시티 안전 플랫폼

사진 한 장으로 도시 위험을 신고하면, AI가 종류를 판별하고 안전신문고식 신고서를 자동 작성해 지자체 관제로 연결합니다.

▶ 시민 신고 데모 🏛 관제 대시보드 소스 코드(GitHub)

1. 무엇을 푸는가

시민 관점

“텍스트 없는 3초 신고” — 사진만 올리면 분야·제목·내용이 자동 작성되고, 접수번호로 진행을 확인.

지자체 관점

자동 분류·우선순위·중복 묶음으로 민원 분류·처리 시간을 단축. 사진·AI 요약을 한 화면에서 처리.

3. 동작 흐름

1사진 신고촬영/업로드 + 위치
→
2AI 분류분야·세부유형 판별
→
3신고서 자동작성제목·내용·일시 채움
→
4지자체 처리관제·상태·공문서

4. 기능 현황 (기술용어 제외)

됨 지금 되는 것

  • 사진으로 신고
  • 사진 보고 위험 종류 자동 판별
  • 분야별 맞춤 신고서 AI 자동작성
  • 위치 3가지(사진·현재위치·주소검색)
  • 접수번호·진행 상태 조회
  • 관할 자동판별·중복 묶음
  • 관제: 목록·사진·상태처리·공문서 초안

일부 부분만

  • 사진 판별 정확도 — 보이는 건(포트홀·쓰레기·침수) 잘, 신호위반·과속은 약함→수정
  • 공문서 자동작성 — 값 채우기 수준

아직 안 되는 것

  • 대화형 안내(챗봇) — 정해진 문구만
  • 얼굴·차량번호 자동 가리기
  • 처리 상황 알림(문자·앱)
  • 실제 안전신문고 자동 전송
  • 허위·도배 신고 차단
  • 개인정보·회원관리(의도적 제외)

5. 분류 체계 (안전신문고 3대 분야)

자동차·교통 위반

불법 주·정차 / 교통법규 / 번호판 / 불법 튜닝

생활불편

쓰레기 무단투기 / 불법 광고물 / 자전거·이륜차 방치 / 기타

안전

도로·시설물 파손 / 침수 / 대기·수질 오염 / 소방안전 / 기타

6. 공공 서비스 전환 시 보안 설계

서버 공격 방어

웹방화벽·DDoS 차단·과부하 대기열 · KISA 사이버대피소(무료)

본인확인

간편인증·PASS 위임 → 안전키패드 직접 구현 불필요

개인정보

익명·최소수집·비식별(블러)·암호화·자동 파기

운영 기반

CSAP 인증 클라우드 · ISMS-P · 개인정보 영향평가(PIA)

7. 향후 로드맵

  • 1단계 관리형 DB 전환 + 본인확인(간편인증) 연계
  • 2단계 보안 강화(방화벽·DDoS·이상탐지) + 얼굴/번호판 실블러
  • 3단계 보안 인증(ISMS-P) + 개인정보 영향평가(PIA)
  • 4단계 CSAP 인증 클라우드 입주 + 안전신문고/지자체 연계
  • 5단계 청주 실증 → 다수 지자체 확산(SaaS)
